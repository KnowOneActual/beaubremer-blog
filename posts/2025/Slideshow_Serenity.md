We have all been there. You are minutes away from starting a corporate event. Someone hands you a flash drive full of
photos. They ask, "Can we loop these on the big screen?"

That quiet panic is common in the event world. Fortunately, you can avoid this stress. Automating the process saves you from inserting, resizing, and centering every image by hand. Let's look at two ways to turn a chaotic folder of photos into a polished PowerPoint slideshow.

### Method 1: The Ultimate Automation with Python

If you do this often, a Python script is your best friend. It is fast, reusable, and does the hard work for you. To make
this easy, I wrote a script called **SlideWrangler**.

**Why use a script?**

- **It is fast.** It processes hundreds of images in seconds.
- **It is smart.** SlideWrangler fixes sideways photos, skips non-image files, and optimizes them for the screen.
- **It is consistent.** Every slideshow is formatted perfectly, every time.

**How to get it:**

The code and instructions are on GitHub. Visit the repository to get started.

➡️ **[Get SlideWrangler on GitHub](https://github.com/KnowOneActual/SlideWrangler)**

The README file walks you through the simple setup. In a few minutes, you will have a tool ready to handle any photo
dump.

### Method 2: The Quick Fix Inside PowerPoint

If you do not want to use a script, PowerPoint has a handy built-in feature. It works wonders for a quick fix.

**Why use this method?**

- **No setup.** You only need PowerPoint.
- **It is visual.** You can see what you are doing at each step.
- **It is fast.** It is much quicker than adding photos one by one.

**Here's how you do it:**

1. Open PowerPoint and click the **Insert** tab.
2. Click **Photo Album** and choose **New Photo Album...**.
3. In the pop-up window, click **File/Disk...**.
4. Find your folder of images. Select them all (press Ctrl+A) and click **Insert**.
5. In the Photo Album window, you can adjust settings. Set **Picture layout** to **Fit to slide**.
6. Click **Create**. PowerPoint generates a presentation with one photo per slide.

### Which Method is Right for You?

If you run events or handle slides often, setting up **SlideWrangler** is worth the time. For a quick, one-off task, the
built-in Photo Album feature works great.

Either way, you can now handle last-minute photo requests with confidence. No more frantic clicking. Your slideshow is
ready for the show.
